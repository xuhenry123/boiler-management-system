## 第一章 系统概述

### 1.1 项目背景

随着城市化进程的加速和人们对居住环境品质要求的不断提升，集中供热已成为北方城市冬季供暖的主要方式。传统的锅炉集中供热系统主要依赖人工经验和简单仪表进行管理，存在以下显著问题：

**能源利用效率低下**：传统系统无法根据实际需求动态调整供热参数，导致能源浪费严重，供热效率通常仅为60%-70%。

**供热不均衡**：由于管网设计不合理和缺乏有效的平衡调节手段，不同区域、不同楼栋之间温差明显，用户投诉率居高不下。

**自动化程度低**：大量操作依赖人工完成，无法实现精细化管理和远程监控，运维成本高昂。

**响应迟缓**：面对用户投诉或设备故障，系统响应速度慢，问题处理周期长。

本项目旨在构建一套智能化、数字化、可视化的锅炉集中供热管理系统，通过物联网感知技术、人工智能算法和大数据分析，实现对供热全过程的精确控制和优化调度。

### 1.2 项目目标

本系统的建设目标包括：

1. **提高供热效率**：通过智能控制算法和优化调度，将锅炉运行效率提升至85%以上，降低能源消耗15%-20%。

2. **实现精准供热**：基于用户需求、室内温度反馈和气象数据，实现按需供热、动态调节，确保室温达标率≥98%。

3. **保障系统安全**：建立完善的监控预警机制，实现对温度、压力、流量等关键参数的实时监测和异常预警。

4. **降低运维成本**：通过自动化控制和远程监控，减少现场人工干预，降低运维成本30%以上。

5. **提升服务质量**：建立用户反馈渠道，实现问题快速响应，提升用户满意度至90%以上。

**（4）按需供热控制功能**

基于采集数据、预测结果和用户画像，系统实现智能按需供热控制：

控制策略：
- **离家降负荷**：检测到用户离家模式，自动降低供热温度
- **回家预热**：根据用户回家时间预测，提前调整供热参数
- **夜间低温**：在保证室内温度不低于设定值的前提下，夜间降低供水温度
- **天气补偿**：根据室外温度变化，自动调整供热曲线

## 第四章 核心算法设计

### 4.1 PID控制算法

#### 4.1.1 算法原理

PID（Proportional-Integral-Derivative）控制是工业控制中最广泛应用的控制算法之一。其基本原理是将系统的设定值与实际测量值进行比较，得到偏差信号，然后根据偏差的比例、积分和微分三个部分进行运算，产生控制信号，驱动执行器动作，使系统输出跟踪设定值。

PID控制器的数学表达式为：

```
u(t) = Kp × e(t) + Ki × ∫e(t)dt + Kd × de(t)/dt
```

其中：
- u(t)：控制输出
- e(t)：偏差信号（设定值 - 测量值）
- Kp：比例系数
- Ki：积分系数
- Kd：微分系数

**各参数作用**：

| 参数 | 作用 | 调大效果 | 调小效果 |
|------|------|---------|---------|
| Kp | 加快响应 | 响应加快，消除偏差加快 | 响应变慢 |
| Ki | 消除稳态误差 | 消除稳态误差，但可能超调 | 稳态误差消除慢 |
| Kd | 预测未来 | 抑制超调，提高稳定性 | 超调增大 |

#### 4.1.2 经典PID实现

系统实现的经典PID控制算法流程图：


**Java实现代码片段**：

```java
public Map<String, Object> calculate(double measuredValue) {
    long currentTime = System.currentTimeMillis();
    double dt = (currentTime - lastTime) / 1000.0;
    
    // 计算偏差
    double error = setpoint - measuredValue;
    
    // 积分环节（带积分限幅）
    integral += error * dt;
    if (integral > outputMax) {
        integral = outputMax;
    } else if (integral < outputMin) {
        integral = outputMin;
    }
    
    // 微分环节
    double derivative = (error - lastError) / dt;
    
    // PID输出计算
    double output = kp * error + ki * integral + kd * derivative;
    
    // 输出限幅
    if (output > outputMax) {
        output = outputMax;
    } else if (output < outputMin) {
        output = outputMin;
    }
    
    // 更新历史值
    lastError = error;
    lastTime = currentTime;
    
    return result;
}
```

```java
private double[] calculateKpAdjustment(double error, double deltaError) {
    double adjustment = 1.0;

        adjustment = 1.2;    // 误差大时增大Kp加快响应
```

### 4.2 温度预测算法

#### 4.2.1 算法原理

温度预测是实现按需供热的关键技术，系统采用LSTM（Long Short-Term Memory）长短期记忆网络进行温度预测。LSTM是一种特殊的循环神经网络（RNN），能够学习长期依赖信息，非常适合时间序列预测任务。

**LSTM网络结构**：

```
┌─────────────────────────────────────────────────────────┐
│                  LSTM网络结构示意图                       │
├─────────────────────────────────────────────────────────┤
```

**LSTM核心公式**：

```
遗忘门: f_t = σ(W_f · [h_{t-1}, x_t] + b_f)
输入门: i_t = σ(W_i · [h_{t-1}, x_t] + b_i)
        C~_t = tanh(W_c · [h_{t-1}, x_t] + b_c)
细胞状态: C_t = f_t × C_{t-1} + i_t × C~_t
输出门: o_t = σ(W_o · [h_{t-1}, x_t] + b_o)
        h_t = o_t × tanh(C_t)
```

### 4.3 二网平衡优化算法

#### 4.3.1 问题描述

二次管网平衡优化的目标是调整各建筑物的阀门开度，使各建筑物的实际流量与其需求流量一致，从而实现管网平衡运行。

**优化目标函数**：

```
min F(x) = Σ|w_i - w_i_target| × flow_i

其中：
x = [w_1, w_2, ..., w_n] - 阀门开度向量
w_i - 第i个阀门开度 (0 ≤ w_i ≤ 1)
flow_i - 第i个建筑物的流量
w_i_target - 第i个阀门的理想开度
```

**约束条件**：
- 阀门开度限制：0 ≤ w_i ≤ 1
- 节点流量平衡：Σflow_i = flow_total
- 压力约束：P_min ≤ P_i ≤ P_max

```java
public Map<String, Object> optimizeWithGA(Long stationId, 
                                           List<Map<String, Object>> buildings) {
    // 算法参数设置
    int populationSize = 50;
    int maxGenerations = 100;
    double crossoverRate = 0.8;
    double mutationRate = 0.05;

    // 1. 初始化种群
    List<double[]> population = initializePopulation(populationSize, buildings.size());
    
    double bestFitness = Double.MAX_VALUE;
    double[] bestSolution = null;

    // 2. 迭代优化
    for (int generation = 0; generation < maxGenerations; generation++) {
        // 评估适应度
        for (double[] individual : population) {
            double fitness = calculateFitness(individual, buildings);
            if (fitness < bestFitness) {
                bestFitness = fitness;
                bestSolution = individual.clone();
            }
        }
        
        // 进化种群
        population = evolvePopulation(population, buildings, 
                                      crossoverRate, mutationRate);
    }

    // 返回优化结果
    Map<String, Object> result = new HashMap<>();
    result.put("stationId", stationId);
    result.put("optimizationType", "ga");
    result.put("valveOpenRatios", buildValveRatios(buildings, bestSolution));
    result.put("objectiveValue", bestFitness);
    result.put("iterationCount", maxGenerations);
    result.put("status", 1);
    
    return result;
}

// 适应度计算
private double calculateFitness(double[] solution, List<Map<String, Object>> buildings) {
    double totalDeviation = 0;
    for (int i = 0; i < solution.length && i < buildings.size(); i++) {
        double targetTemp = ((Number) buildings.get(i).get("targetTemp")).doubleValue();
        double currentFlow = ((Number) buildings.get(i).get("flow")).doubleValue();
        double ratio = solution[i];
        double expectedFlow = currentFlow * ratio;
        double deviation = Math.abs(expectedFlow - targetTemp * 5);
        totalDeviation += deviation;
    }
    return totalDeviation;
}

// 种群进化
private List<double[]> evolvePopulation(List<double[]> population, 
                                          List<Map<String, Object>> buildings,
                                          double crossoverRate, 
                                          double mutationRate) {
    List<double[]> newPopulation = new ArrayList<>();
    
    // 精英保留: 保留top 10%
    Collections.sort(population, (a, b) -> 
        Double.compare(calculateFitness(a, buildings), 
                       calculateFitness(b, buildings)));
    int eliteCount = population.size() / 10;
    for (int i = 0; i < eliteCount; i++) {
        newPopulation.add(population.get(i).clone());
    }

    // 生成新个体
    while (newPopulation.size() < population.size()) {
        // 锦标赛选择
        double[] parent1 = tournamentSelection(population, buildings);
        double[] parent2 = tournamentSelection(population, buildings);
        
        double[] offspring;
        
        // 交叉
        if (Math.random() < crossoverRate) {
            offspring = crossover(parent1, parent2);
        } else {
            offspring = parent1.clone();
        }
        
        // 变异
        if (Math.random() < mutationRate) {
            mutate(offspring);
        }
        
        newPopulation.add(offspring);
    }
    
    return newPopulation;
}
```

**算法参数**：
- 惯性权重 w = 0.7：保持粒子运动惯性
- 学习因子 c1 = 1.5：个体认知能力
- 学习因子 c2 = 1.5：群体协作能力

```java
public Map<String, Object> optimizeWithPSO(Long stationId, 
                                           List<Map<String, Object>> buildings) {
    int particleCount = 30;
    int maxIterations = 100;
    double w = 0.7;
    double c1 = 1.5;
    double c2 = 1.5;

    // 初始化粒子群
    List<double[]> particles = new ArrayList<>();
    List<double[]> velocities = new ArrayList<>();
    List<double[]> bestPositions = new ArrayList<>();
    double[] globalBest = null;
    double globalBestFitness = Double.MAX_VALUE;

    for (int i = 0; i < particleCount; i++) {
        double[] particle = new double[buildings.size()];
        for (int j = 0; j < buildings.size(); j++) {
            particle[j] = Math.random();  // 随机初始化位置
        }
        particles.add(particle);
        velocities.add(new double[buildings.size()]);
        bestPositions.add(particle.clone());

        // 计算初始适应度
        double fitness = calculateFitness(particle, buildings);
        if (fitness < globalBestFitness) {
            globalBestFitness = fitness;
            globalBest = particle.clone();
        }
    }

    // 迭代优化
    for (int iter = 0; iter < maxIterations; iter++) {
        for (int i = 0; i < particleCount; i++) {
            // 更新速度
            for (int j = 0; j < buildings.size(); j++) {
                double r1 = Math.random();
                double r2 = Math.random();
                velocities.get(i)[j] = w * velocities.get(i)[j] +
                    c1 * r1 * (bestPositions.get(i)[j] - particles.get(i)[j]) +
                    c2 * r2 * (globalBest[j] - particles.get(i)[j]);
                
                // 更新位置
                particles.get(i)[j] += velocities.get(i)[j];
                particles.get(i)[j] = Math.max(0, Math.min(1, particles.get(i)[j]));
            }

            // 更新适应度
            double fitness = calculateFitness(particles.get(i), buildings);
            if (fitness < calculateFitness(bestPositions.get(i), buildings)) {
                bestPositions.set(i, particles.get(i).clone());
            }
            if (fitness < globalBestFitness) {
                globalBestFitness = fitness;
                globalBest = particles.get(i).clone();
            }
        }
    }

    return result;
}
```

### 4.4 水力模拟算法

#### 4.4.1 算法原理

水力模拟是基于质量守恒和能量守恒原理，通过迭代计算求解管网中各节点的压力和流量分布。

**基本方程**：

1. **节点流量平衡方程**：
```
ΣQ_in = ΣQ_out
```

2. **管段压降方程**（沿程阻力）：
```
ΔP = f × (L/D) × (ρv²/2)
简化: ΔP = R × Q²
```

其中：
- R：管段阻力系数
- Q：流量
- ΔP：压降

3. **压力迭代方程**：
```
P_new = P_old + ΔP_iteration
```

```java
public Map<String, Object> simulate(Map<String, Object> networkConfig) {
    List<Map<String, Object>> nodes = (List<Map<String, Object>>) networkConfig.get("nodes");
    List<Map<String, Object>> segments = (List<Map<String, Object>>) networkConfig.get("segments");
    
    // 初始化节点压力
    Map<String, Double> nodePressures = initializePressures(nodes);
    Map<String, Double> segmentFlows = new HashMap<>();
    
    // 迭代计算
    boolean converged = false;
    int maxIterations = 100;
    int iteration = 0;
    
    while (!converged && iteration < maxIterations) {
        Map<String, Double> oldPressures = new HashMap<>(nodePressures);
        
        // 计算管段流量
        for (Map<String, Object> segment : segments) {
            String startNode = (String) segment.get("startNode");
            String endNode = (String) segment.get("endNode");
            double flow = calculateFlow(
                nodePressures.get(startNode) - nodePressures.get(endNode),
                (double) segment.getOrDefault("resistance", 0.01)
            );
            segmentFlows.put((String) segment.get("segmentCode"), flow);
        }
        
        // 计算节点压力
        for (Map<String, Object> node : nodes) {
            String nodeCode = (String) node.get("nodeCode");
            if (!"source".equals(node.get("nodeType"))) {
                double pressure = calculateNodePressure(node, nodePressures, segmentFlows);
                nodePressures.put(nodeCode, pressure);
            }
        }
        
        // 检查收敛
        double maxChange = calculateMaxChange(oldPressures, nodePressures);
        converged = maxChange < 0.001;
        iteration++;
    }
    
    return result;
}

// 流量计算
private double calculateFlow(double pressureDiff, double resistance) {
    if (Math.abs(pressureDiff) < 0.0001) {
        return 0;
    }
    return Math.signum(pressureDiff) * Math.sqrt(Math.abs(pressureDiff) / resistance);
}

// 节点压力计算
private double calculateNodePressure(Map<String, Object> node, 
                                     Map<String, Double> nodePressures,
                                     Map<String, Double> segmentFlows) {
    double demand = ((Number) node.getOrDefault("demand", 0.0)).doubleValue();
    double elevation = ((Number) node.getOrDefault("elevation", 0.0)).doubleValue();
    
    // 静压 + 动压修正
    double pressureBase = GRAVITY * elevation * WATER_DENSITY / 1000000.0;
    double flowContribution = demand * 0.01;
    
    return Math.max(0.1, pressureBase + flowContribution);
}
```
