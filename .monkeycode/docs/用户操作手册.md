## 1. 系统概述

### 1.1 系统简介

锅炉集中供热智慧管理系统是一套基于物联网、大数据和人工智能技术的智慧供热管理平台。系统实现了对供热全流程的监控、分析、优化和控制，帮助供热企业提高运行效率、降低能耗、保证供热质量。

### 1.2 系统架构

系统采用前后端分离架构：
- **前端**：Vue 3 + Element Plus + ECharts
- **后端**：Spring Boot + MyBatis Plus + MySQL
- **技术特点**：响应式设计、可视化图表、实时数据监控

### 1.3 浏览器要求

推荐使用以下浏览器访问系统：
- Chrome 80+
- Firefox 75+
- Edge 80+
- Safari 13+

## 3. 一级功能模块使用说明

### 3.1 监控大屏

**路径**：`/dashboard`

**功能描述**：展示系统的整体运行状态，包括换热站数量、建筑物数量、热用户数量、告警数量等关键统计数据，以及温度趋势图、换热站状态图、热负荷曲线等可视化图表。

#### 功能组件说明

| 组件名称 | 类型 | 功能说明 |
|---------|------|---------|
| 换热站数量 | 统计卡片 | 显示系统中配置的换热站总数 |
| 建筑物数量 | 统计卡片 | 显示系统中管理的建筑物总数 |
| 热用户数量 | 统计卡片 | 显示系统服务的热用户总数 |
| 告警数量 | 统计卡片 | 显示当前未处理的告警数量 |
| 温度趋势监控 | 折线图 | 展示一次侧供水/回水、二次侧供水/回水的温度变化趋势 |
| 换热站运行状态 | 饼图 | 显示运行中、停止、故障的换热站数量分布 |
| 热负荷曲线 | 柱状图 | 展示各时段的供热负荷变化 |
| 最近告警 | 表格 | 显示最近的告警信息列表 |

#### 操作说明

- **刷新数据按钮**：点击可手动刷新所有统计数据和图表数据

## 5. 附录

### 5.1 常用操作指南

#### 数据刷新
大多数页面支持手动刷新数据，点击页面中的"刷新数据"按钮即可更新最新数据。

#### 筛选查询
- 使用下拉框进行类型筛选
- 使用搜索框进行关键字搜索
- 使用时间选择器进行时间范围筛选

#### 图表交互
- 鼠标悬停查看详细数据
- 点击图例可以显示/隐藏特定数据系列
- 支持图表放大、导出等操作

### 5.2 系统维护说明

#### 告警处理流程
1. 在告警管理页面查看告警信息
2. 点击"确认"按钮确认告警
3. 分析告警原因
4. 点击"处理"按钮，填写处理方式
5. 执行相应处理操作

#### 设备维护流程
1. 在设备管理页面查看设备状态
2. 对需要维护的设备点击"维护"按钮
3. 记录维护情况
4. 维护完成后更新设备状态
